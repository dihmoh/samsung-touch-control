# Samsung Touch Control

Utilitário leve para controlar o touchscreen do Samsung Galaxy Book 3 360.

## Funcionalidades
- Ícone na bandeja do sistema com estado ON/OFF
- Ativar, desativar e alternar o touchscreen
- Tecla de atalho global (Ctrl+Alt+T)
- Popup de notificação ao mudar estado
- Iniciar automaticamente com o Windows
- Painel com botão ON/OFF

## Requisitos
- Windows 11
- Samsung Galaxy Book 3 360
- .NET 10.0

## Como usar
1. Execute o instalador `SamsungTouchControl_Setup.exe`
2. O app inicia automaticamente na bandeja
3. Clique com botão direito no ícone para o menu
4. Use **Ctrl+Alt+T** para alternar o touch
